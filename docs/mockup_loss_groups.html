JKAN Loss Display Mockup Comparison

Context: DesInventar datasets contain 50-136 individual loss records per country. The current JKAN mapper concatenates all loss descriptions into a single comma-separated string, producing an unreadable wall of text.

Dataset: Albania (DesInventar) — 111 loss records across 9 hazard types, 4,659 disaster events (1973-2015).

This mockup shows the current state and three proposed alternatives. Click the collapsible sections to test interactivity.
